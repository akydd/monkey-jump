# Monkey Jump

A vertical platform game where you guide a monkey up a towering tree by leaping between branches. How high can you climb before the branches give way beneath you?

## Gameplay

Branches break after you stand on them too long — they flash orange as a warning. Keep moving upward to survive. The game ends if you fall to the bottom.

Your best height (in metres) is tracked at the top of the screen.

### Controls

| Key | Action |
|-----|--------|
| `←` / `→` | Move left / right |
| `↑` | Jump |

## Running locally

```bash
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.
